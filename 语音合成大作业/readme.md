#### 大作业为语音合成  
##### pre中存放ppt与讲稿  ：
仅供参考
##### 文件处理中存放处理用python文件  ：
用于生成ljspeech格式的数据集  
其中：  
main用于从txt转化为csv  
trans用于处理txt格式

##### 想要自制数据集可以按以下步骤处理：
1. 下载视频文件  
2. 用pr或剪映从视频中提取音频，保存为.wav格式
3. 使用pr2022智能字幕插件或者剪映自带的生成字幕功能，生成.srt格式字幕
4. 导出字幕文件，保存为txt格式
5. 可以对字幕进行人工校对 
6. 运行main.py，会自动将wav文件按字幕切分成小段音频
7. 运行trans.py，将字幕txt文件处理成与ljspeech数据集相同的标注格式  
（如果已经有字幕与音频文件，跳过123步）
